# ControlHorarioApp
Aplicación Android para control horario basado en GPS con ajuste automático según turnos.
## Funcionalidades
- Detecta entrada/salida del área de trabajo mediante GPS.
- Ajusta hora según turno (mañana/tarde).
- Guarda registros en SQLite.
- Exporta registros a CSV para Excel.
- Notificaciones al detectar entrada/salida.
- Pantalla de configuración para coordenadas y turnos.
